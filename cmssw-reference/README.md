# CMSSW Reference Subset

Small subset of upstream CMSSW/Dfinder/ZDC files that were directly inspected,
wrapped, or modified during the reproduction.

This is not a vendor copy of CMSSW. It exists so the analysis-code archive
records which upstream files mattered for the reproduction.

- `Bfinder/Bfinder/src/Dfinder.cc`
- `HeavyIonsAnalysis/ZDCAnalysis/python/ZDCAnalyzersHC2023_cff.py`
- `HeavyIonsAnalysis/ZDCAnalysis/src/ZDCHardCodeHelper.*`
