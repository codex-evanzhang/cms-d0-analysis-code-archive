# Official MC Fragments

This folder contains code/config fragments rendered from stored CMS McM
metadata for the official D0 UPC MC setup.

Included:

- `python/`: cleaned CMSSW fragment files used by the local MC smoke workflow.
- `raw/`: raw fragment text captured from the McM snapshots.

Excluded:

- generated MC events,
- MINIAODSIM/AODSIM outputs,
- ROOT forests,
- CRAB/runtime products.
