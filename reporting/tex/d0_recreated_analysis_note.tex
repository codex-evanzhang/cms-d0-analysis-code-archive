\section*{5. Offline Event Selection}
\textbf{Status:} strong partial\\
\textbf{Purpose in AN:} Recreate event-level UPC selection: PV/MET filters, ZDC 0nXn selection, and HF rapidity gap.
\paragraph{Reproduced or available now.}
\begin{itemize}
\item Nominal ZDC selection now follows the AN calibrated offline recipe: ZDCsignal = HADsum + 0.1 EMsum, with thresholds plus > 1100 GeV and minus > 1000 GeV.
\item The 0nXn class is implemented as an XOR: plus-side Xn with minus-side 0n, or minus-side Xn with plus-side 0n.
\item EmptyBX events validate the 0n-side noise tail, while fitted 1n/2n/3n ZDC spectra validate the Xn-side efficiency.
\item The EM-pileup survival probability is treated as a later correction/uncertainty, not as part of the raw XOR cut.
\item HF gap thresholds are derived with low-track pass and high-track leakage checks, with MC used only where valid.
\item Event cutflow exists for the two-file CRAB smoke and for the 98-forest selected skim.
\item AN-proxy event plots are generated from the reproduced 98-forest Section 5 selected event tree.
\end{itemize}
\paragraph{Still missing or not final.}
\begin{itemize}
\item Full AN event efficiency tables, final ZDC Xn integration, 0n fake-rate accounting, and EM-pileup correction tables still need a full-statistics pass.
\end{itemize}
